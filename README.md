# CryptRaider

Developed with Unreal Engine 5


## Aim
Complete the puzzle by stealing a treasure from a crypt.
The game currently has only 1 level.


## Control
WASD - Move.
LMB (Hold) - Carry a prop.
LMB (Release) - Drop a prop.
Spacebar - Jump.
